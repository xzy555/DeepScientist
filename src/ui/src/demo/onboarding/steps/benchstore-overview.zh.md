BenchStore 是“先选一个 benchmark，再开始 Project”的地方。

最实用的用法是：

1. 先从 **Recommended** 开始
2. 打开一个 benchmark 详情页
3. 判断它是否适合你的机器和兴趣
4. 如果合适，就用 `Start` 把它送进项目启动表单

可以这样理解这几个区域：

- **Store** = 浏览可选 benchmark
- **Library** = 回看你已经收藏或启动过的任务
- **Recommended** = 第一次选择时最快的入口

提示：不要试图一次看完所有任务。先挑一个推荐项，打开详情页，然后做一次明确的选择。
