This detail page is where you decide whether this benchmark should become your next project.

For a beginner, read it in this order:

1. **compatibility**: can this machine handle it
2. **task description**: what work it is actually asking for
3. **risk / install state**: is it ready enough to continue
4. **linked quests**: has this task already been used before

After reading that:

- if it looks suitable and already installed, click `Start`
- if local assets are still missing, click `Download` first
- if the local install looks broken or incomplete, use `Reinstall`
