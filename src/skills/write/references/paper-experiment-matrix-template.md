# Paper Experiment Matrix Template

Use this template when the experiments section needs one compact planning and reporting surface.

## Recommended columns

- `exp_id`
- `title`
- `tier`
- `experiment_type`
- `status`
- `feasibility_now`
- `claim_ids`
- `research_question`
- `metrics`
- `paper_placement`
- `next_action`

## Review rule

Keep the matrix aligned with `paper/selected_outline.json` and `paper/evidence_ledger.json`.
