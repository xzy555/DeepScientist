# Outline-Evidence Contract Example

Use this as a compact example when a paper line needs one durable bridge between outline sections and evidence.

## Minimal shape

- `story`
- `research_questions`
- `contributions`
- section list with:
  - `section_id`
  - `paper_role`
  - linked claims
  - required evidence
  - optional evidence

## Good outcome

Every main-text claim can point to one durable artifact path or one explicit gap.
